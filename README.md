# ZUNAJRAVNINSKI GRAFI (OUTERPLANAR GRAPHS)
**AVTORJA:** Jure Kraševec in Urh Videčnik

# Opis projektne naloge
Pri predmetu *Finančni praktikum* sva moral narediti projektno nalogo, ki je temeljila na naslednjem problemu:
\\
Ali za zunajravninski graf G, z |V(G)| = n velja, da je velikost njegove lihe neodvisne množice večja ali neaka kot n/7.
Projektna naloga je bila sestavljena iz 2 delov. 
Prvi del je bila implementacija funkcije, ki preveri ali je graf zunajravninski in nato sistematično preverjanje zgoraj navedenega problema.
Drugi del pa je bila implementacija funkcije, ki naključno naredi zunajravninski graf in nato zanj preveri problem.
